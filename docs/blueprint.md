# **App Name**: Scriptify Studio

## Core Features:

- Integrated Creation Suite: Centralized editor for influencer traits, video scene properties and AI script generation.
- Influencer Image Analysis: Use uploaded reference images of an influencer to extract character traits like name, niche, personality, and appearance.
- Text-to-Profile AI Tool: Paste reference text about a persona and automatically extract and pre-fill the data, in Portuguese, using generative AI. A tool helps ensure no keys are omitted, so the result is a valid JSON object.
- Scene Background Analysis: Use generative AI to propose video settings, moods and backgrounds based on uploaded reference images. All content is translated to Brazillian Portuguese.
- AI product description: Analyze an uploaded product image, automatically determine a complete product description to streamline content creation, translating into Brazillian Portuguese
- AI-Powered Script Generation: Leverage a scene’s properties, and the influencer traits to automatically generate a relevant and coherent script.
- Formatted Script Export (JSON or MarkDown): A structured format helps facilitate copy-pasting and automation.

## Style Guidelines:

- Background color: Soft Lavender (#F4F0F8) for a calming and creative atmosphere.
- Primary color: Deep Purple (#6C4AB6) to signify creativity and innovation.
- Accent color: Dusty Pink (#E8A0BF) for highlights and calls to action.
- Headline font: 'Space Grotesk' sans-serif for a modern, computerized feel, suitable for titles.
- Body font: 'Inter' sans-serif, for a clean, objective, readable text.
- Use a set of consistent, modern icons, in the style of smooth line art.
- Maintain a clean, spacious layout that is easy to scan and use.
